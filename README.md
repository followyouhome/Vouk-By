### Setup
**Prerequisites:** Node.js 0.10+ and MongoDB v2.4+
1. Start MongoDB instance  
   `mongod --dbpath <path>`
2. Install dependencies  
   `npm install`
3. Run server  
   `npm start` (or `node keystone`)
4. Navigate to the app  
   `http://localhost:3000/`
